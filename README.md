# CodeAlpha Object Detection

This project uses YOLOv8 for real-time object detection and tracking with a webcam.

## Features
- Detects and tracks objects live from the camera feed
- Displays bounding boxes and tracking IDs
- Uses the lightweight YOLOv8 Nano model

## Requirements
Install the required packages:

```bash
pip install ultralytics opencv-python
```

## Run
```bash
python app.py
```

Press `q` to quit the webcam window.
